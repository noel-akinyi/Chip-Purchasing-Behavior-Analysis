# Chip Purchasing Behavior Analysis
### *Data-Driven Strategic Recommendations for Retail Category Review*

## Project Objective
The goal of this analysis is to evaluate chip product purchasing data to provide a Retail Manager with strategic recommendations. By focusing on **customer segments** and their specific buying behaviors, this project identifies growth opportunities for an upcoming category review.

---

## 📊 Key Business Questions
* **Who are the high-value customers?** (Which segments spend the most?)
* **What are they buying?** (Preferences in brand, pack size, and frequency.)
* **Where is the growth?** (Identifying under-served segments or popular trends.)

---

## The Data & Scope
* **Dataset:** Transactional retail data and customer demographic segmentation.
* **Scope:** Analysis is strictly limited to **chip products** to ensure category-specific insights.

---

## 🛠️ Tech Stack & Libraries
This analysis was built using **Python 3.10.19** and the following data science ecosystem:

* **Data Manipulation:** `Pandas`, `NumPy` (for cleaning and structuring retail transactions).
* **Data Quality & Cleaning:** `re` (Regex/Regular Expressions for product name standardization).
* **Statistical Analysis:** `Scipy.stats` (to validate behavioral trends between segments).
* **Visualization:** `Matplotlib`, `Seaborn` (for creating the Retail Manager's dashboard).
* **Environment:** `Jupyter Notebooks` (for reproducible, step-by-step analysis).
